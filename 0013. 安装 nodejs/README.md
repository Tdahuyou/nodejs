**查看 node 版本 - 验证 nodejs 是否成功安装**

无论是从旧版的页面还是新版的页面下载的 nodejs，最终验证安装成功的命令（查看 node 版本的命令）都是一致的 `node -v`。

```bash
$ node -v
# v20.10.0
```
